set容器简介

2016.05.27 ryl STL

先上彩蛋：

  1. cplusplus
  2. cppreference
  3. stackoverflow
    #include<set>头文件内定义了两个有序的hash容器set和multiset

set简介

  1. 以特定顺序存储唯一元素；
  2. 键值就是值本身；
  3. 容器中的元素都是不可修改的(都是const类型)，但是可以insert，remove；
  4. 容器通过二叉搜索树实现。

set使用

  1. Subsets II(Recursion)

insert()

set中没有则插入，返回一个pair对象，pair.first()指向插入的元素或set中已有的该元素，pair.second()指示该元素是已有的还是新插入的。
